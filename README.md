# gw-notification

## Описание проекта
`gw-notification` — это микросервис для обработки и сохранения информации о крупных денежных переводах. Сервис получает сообщения о переводах крупных сумм (от 30 000) через брокер сообщений Kafka и сохраняет историю операций в MongoDB для дальнейшей аналитики и мониторинга.

## Основные функции
1. Получение сообщений о переводах крупных денежных сумм от сервиса кошелька через Kafka.
2. Сохранение полученных данных в MongoDB.
3. Возможность последующего расширения логики обработки сообщений (например, уведомления или аналитика).

### Сборка

```bash
go build -o gw-notification ./cmd/main.go
```

### Запуск

1. Настройте файл конфигурации `config.env` или задайте переменные окружения (см. ниже).
2. Запустите сервис:

```bash
./gw-notification -c ./config.env
```

## Поддерживаемые валюты

Сервис поддерживает любые валюты через расширение списка в базе данных.

### Окружение

Параметры подключения к базе данных, порты и другие настройки задаются через файл `config.env` или переменные окружения:

| Переменная окружения | Описание                 |
|----------------------|--------------------------|
| `SERVER_PORT`        | Порт для запуска сервера |
| `DATABASE_HOST`      | Хост базы данных         |
| `DATABASE_PORT`      | Порт базы данных         |
| `DATABASE_USER`      | Пользователь базы данных |
| `DATABASE_PASSWORD`  | Пароль базы данных       |
| `DATABASE_NAME`      | Имя базы данных          |
| `EVENT_BUS_HOST`     | Хост очереди сообщений   |
| `EVENT_BUS_PORT`     | Порт очереди сообщений   |
